# Function: Start()

> **Start**(`instance`, `appid`?, `title`?, `width`?, `height`?): `Promise`\<`void`\>

启动应用注册表中包含的指定应用。

## Parameters

• **instance**: [`Ploy3D`](../classes/Ploy3D.md)

引擎实例。

• **appid?**: `string`

所要启动的应用ID。

• **title?**: `string`

应用主窗口标题。

• **width?**: `number`

应用主窗口初始宽度。

• **height?**: `number`

应用主窗口初始高度。

## Returns

`Promise`\<`void`\>

返回事件协程。
